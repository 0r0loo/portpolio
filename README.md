trpc 를 해보려고 했는데 모노레포를 구현하라고 써있어서
모노레포 구현 해보는 김에 내 포트폴리용 레포로 만들면 괜찮을 꺼같아서 만듬


# 명령어
```bash
yarn common  공용 함수 
yarn components  공용 컴포넌트
yarn nextapp  테스트용 넥스트 
yarn nestapp  테스트용 네스트
```

# 폴더구조
backend : 백엔드 프로젝트 관리
frontend : 프론트엔드 프로젝트 관리
lib : 공용으로 쓸 것들
